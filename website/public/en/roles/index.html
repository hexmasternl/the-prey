Roles

The three roles in The Prey: Game Creator, Hunter, and Prey. Learn what each one does before the game starts.

There are three roles in The Prey. Each game is set up by a Game Creator, has exactly one Hunter, and one or more Prey.

Select your role below to learn what it means and how to play it.

◈
Game Creator
The Game Creator role in The Prey: define the playfield, share the join code, assign Hunter and Prey roles, and start the game.
⬤
Hunter
Playing as a Hunter in The Prey: you are the game's single Hunter. Wait out the head start, track every Prey by GPS pins, close the gap, and tag them.
◎
Prey
Playing as Prey in The Prey: use the head start to scatter and hide, keep moving during the hunt, manage background GPS, and survive until the timer ends.
